.. _install:

Install
*******

Please install via only one of the options below:

Via Pip
~~~~~~~

.. code-block:: bash

    pip install aneris-iamc

From Source
~~~~~~~~~~~

.. code-block:: bash

    pip install -r requirements.txt && python setup.py install
